# 課題・カレンダー管理アプリ

大学の課題や提出期限を管理するためのWebアプリです。

## 公開URL

https://yuusep16-tech.github.io/assignment-app/

## スクリーンショット

![アプリ画面](./images/screenshot.png)

## 主な機能

- 教科登録
- 教科ごとの提出曜日設定
- 課題名テンプレート登録
- 提出期限の自動提案
- 課題の追加・編集・削除
- 完了 / 未完了の切り替え
- カレンダー表示
- カレンダー上の課題を押して完了切り替え
- PWA対応
- スマートフォン表示対応

## 開発目的

大学の課題や提出期限を忘れないようにするために開発しています。

教科ごとに提出曜日が決まっている授業が多いため、教科を登録すると次の提出日が自動で提案されるようにしています。また、課題を削除するだけでなく、完了済みとして管理できるようにすることで、何が終わっていて何が残っているかを分かりやすくしています。

## 今後追加したい機能

- 今日やることリスト
- スマートフォンのホーム画面から今日の課題を確認できる機能
- 友達と課題を共有する機能
- PDF / Wordファイルから課題情報を読み取る機能
- 複数人で同じ授業の課題を管理できる機能

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages
- PWA

## 今後の目標

現在は個人用の課題管理アプリとして開発していますが、将来的には友達と課題を共有できるアプリに発展させたいです。

同じ授業を受けている学生同士で、課題名、教科、提出期限、メモなどを共有できるようにし、共通の課題を一緒に管理できる仕組みを作りたいと考えています。
